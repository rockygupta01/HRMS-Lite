# Frontend (HRMS Lite)

React + Vite frontend for HRMS Lite.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Environment

Create `.env` from `.env.example`:

```env
VITE_API_BASE_URL=http://localhost:5001
```

## Netlify

Netlify config lives at repo root in `netlify.toml`.
